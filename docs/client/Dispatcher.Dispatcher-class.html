Class Dispatcher

source code

checkHitAndMove(self, player, playerX, playerY, playerSide, objectX, objectY, objectSide)

source code  
Vérifie si un joueur touche un autre objet
    Si c'est le cas, modifie la position du joueur jusque ce qu'il n'y aie plus contact

Arguments:
    player -- Le joueur concerné.
    playerX -- La coordonnée X du joueur.
    playerY -- La coordonnée Y du joueur.
    playerSide -- La taille du coté d'un joueur.
    objectX -- La coordonnée X de l'objet.
    objectY -- La coordonnée Y de l'objet.
    objectSide -- La taille de l'objet.
